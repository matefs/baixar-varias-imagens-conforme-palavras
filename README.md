## instruções de uso : 

### Requirements: 
```
- python >= 3.0 
```

### Instalação: 
```bash
pip install -U duckduckgo_search
```


#### Baixar imagens: 
./baixar-palavras.sh palavra1 palavra2 palavra3

#### Remover todos os arquivos das imagens não mais utilizados: 


```bash
 find . -type d -name 'images*' -exec rm -r {} +
```
